# 前言（个人经历）

2020年，新型冠状病毒肺炎疫情爆发。大家不得已居家学习办公。疫情在家属实无聊，唯一有兴趣的就是上网冲浪。我也是这一时期，加了一个关于MC的微信群，众所周知，有一些方块人的活力实在是无限的，很多pvp的玩家建立了各种公会，到处招募人才来壮大自己的队伍，然后各个公会就在很多服务器上进行pvp技术的对决。

那个时候，随着公会人数的增加，很多公会自发地形成类似网络虚拟国家的组织（其实就是以群为单位，当然这些在疫情之前也都是存在的）。我也是这个时候接触到了虚拟国，接着是键政圈，再最后就是臭名昭著的“喷系组织”了。
当时网上冲浪冲久了，自己也拉了个群，发展自己的“势力”，其实也发生了大大小小的事。最严重的就是我群被封了三个，之前的微信号被彻底盗走。其实还有很多事情，我就不赘述了。疫情期间的经历让我深刻感受到了网络世界的黑暗一面。

# 虚拟国是什么？

# 网上资料

(萌娘百科)
虚扱国(インターネットミクロネーショ, Internet Micronation),
又称网络微国家，是微国家的一种初级形式。 建立或加入虚拟国是ACG爱好者的常见活动
微国家指的是在一个或多个主权国家的主权范围内或主权范围外单方面宣称自己是一个不属于其他主权国家的独立政治实体[1] 与其他实体有所不同:
不同于马耳他骑士团、梵蒂冈、摩纳哥等微型国家，微国家不被任何主权国家或任何国际组织承认
不同于北塞浦路斯、阿布哈兹、南奥塞梯等有限承认国家，区别同上
不同于澳大利亚原住民、加泰罗尼亚、库尔德等分离主义运动，微国家类似行为艺术
不同于农会、渔会、工会等社会团体，社会团体通常是经过登记的
虚拟国与一般的微国家有所不同：
虚拟国宣称主权范围，但没有实际控制任何领土
虚拟国基于非现实的网络层面，线下活动较少

(虚拟国百科)
虚拟国（virtual state）是一种以百度贴吧或腾讯QQ为平台的，采用真人玩家对抗及合作等形式，拥有丰富可玩的剧情和极其高的自由度，并以号称不氪金便可实现人生“梦想”而著称的超大型社会模拟游戏。虚拟国拥有悠久的历史。国际上，最早的虚拟国开始于1999年，为一群热爱南斯拉夫的外国网友创建。国内的虚拟国发展于建于2006年，甚至更早。不同的虚拟国以不同的时代为背景，设立虚拟“政府”制度，糅合现实历史元素或者架空幻想，主题是综合性的，从基础的外交，游戏，生活，到高精尖的学术讨论交流 。在这里你可以放松心情 ，营造学术范围浓厚，感到温暖的家园，或者体验当时的社会场景，扮演不同时期的不同人员。成为一个总统或者皇帝。

# 所以到底什么是虚拟国

“虚拟国”——也称“网络微国家”（micro-nation / Internet Micronation）——最初是一个比较“纯净”的网络亚文化概念。它指的是这样一种“国”：它没有实际领土，也不被现实国家承认，但在网络空间中由一群志同道合的人构建属于自己的“国度”。成员往往自称“公民”，制定所谓“国歌”“国旗”“宪法”，讨论政治、文化、历史、社会构造……但这本质更像一种“行为艺术”、一种集体幻想／戏谑。

对很多 ACG（动画／游戏／漫画）爱好者来说，“加入虚拟国”是为了寻找归属感、强化社群认同，也是构建自我世界观、获得一种“自治”、“另类现实”的方式。这样设定的好处，是它带来一种“理想国”的氛围 ——— 不用现实国家的法律、不用现实社会的秩序，只靠想象力和共同意志，就可以“建国”。

因此，在虚拟国最初的构想里，它往往是比较浪漫、甚至带着一点乌托邦／理想主义色彩的。很多人是出于好玩、出于对现实不满、出于对另类秩序的好奇，参与其中。

# “键政圈”：网络上的“政治实验场”

与此同时，在许多网络空间，有一股偏重“现实政治与社会议题讨论”的群体——这就是“键政圈”。“键政”本是“键盘政治”的简称，指的是通过键盘/网络对国家、世界、历史、社会制度、意识形态等话题进行评论、争论和讨论的行为。

在某些人看来，“键政”是对现实政治与社会问题的一种关心，是信息时代普通人参与公共议题、表达立场的一种方式。键政圈内部有多种“流派”——从极左到极右都有，从共产主义、无政府主义，到保守主义、新自由主义，再到极端民族主义／原教旨主义。

不过，根据“底室 Wiki”对“键政圈”的描绘：许多键政人士入圈后，会“选一个立场帽子先戴上”，然后将所有异己者“开除”“出局”，开始他们所谓“赛博圣战”：通过网络组织、群体论战、网暴、人肉／DDOS、盗号、封号、炸群乃至“盒子开开”（即人肉搜、隐私曝光）等方式对异见者施压。

长期混迹“键政圈”的人，有可能因为不断对现实进行“键盘化解读”，让自己的思维失衡 —— 变得过于敏感、偏激、阴谋论满天飞，甚至出现“看到什么都想政治化”的倾向。

**不过我个人认为最严重的那些极端分子普及了“盒”武器，让很多人更轻易地就能查到别人的隐私乃至户籍，严重地威胁网络安全。

总的来说，“键政圈” 对一些人来说，是认知与思想的试炼场；对另一些人来说，是陷阱 ——— 心理失衡、情绪激化、现实疏离，害人害己啊。

# 虚拟国 × 键政 = 一场必然会失控的混合物

虚拟国的浪漫 + 键政圈的激进，结合起来会发生什么？

答案是：
创造乌托邦的热情被键政化，最终变成了另一种极端。

典型的演化路径是这样的：

  1. 虚构一个国家 → 想象其政治制度
  2. 成员开始认真讨论 → 变成键政模式的争吵
  3. 立场帽子迅速固化 → 谁都可以被开除国籍
  4. 最终变成另一个政治回声室

于是原本的幻想性世界观变成了：
一群网民在虚拟国里打内战、搞对抗、开盒、挂人。
尤其是开盒挂人，简直可怕啊

虚拟国本是模拟器，结果变成了仿真战场。

# “喷系”的出现与对虚拟国／键政圈的污染

然而，当“喷系”这股力量进入之后，这一切的理想与美好开始被彻底污染。

根据“底室 Wiki”：“喷系”原本和“扣字圈”有些相似，是一种网络辱骂文化——主要靠文字“喷”、用辱骂与煽动戾气取乐、发泄、搞对抗。

后来，喷系吸收了不少来自“黑界”的成员——也就是网络上的“极端对立者”“破坏者”“挑衅者”——并演变成组织化、家族化、门派化。他们开始主动“入侵”其他圈子，用“开盒”“人肉”“查档”“炸群”“封号”“辱骂”“扣字打喷”……等方式，对所谓“异己者”、反对者、不同立场者进行攻击。

在“喷系”进入后，原本带有幻想性质／理想性质的“虚拟国”开始遭到破坏——有些所谓“国民”或“公民”在喷系的影响下，言语极端、戾气十足，与现实中普通人、其他亚文化用户关系紧张。虚拟国的“国策”“理想”“文化认同”，在喷系的“毒蛆式轰炸”下，被扭曲为一种攻击、仇恨、扭曲现实与人际关系的工具。

因此，“虚拟国 + 键政”这个原本或许带有理想主义与乌托邦色彩的网络实验场，变成了某些人的“攻击阵地”、仇恨发泄场、有毒社群。

有资料甚至直接说：“网络虚拟国（虚拟国纯属是被喷系入侵导致的恶俗化）”。

# 死去的乌托邦：从浪漫变恶俗

虚拟国最初是写小说的，
后来变成讨论政治的，
再后来变成打网络仗的，
最后变成互相开骂、开盒、炸群的。

一句话总结：
原本的“网络理想国实验”，
被喷系拖成了“匿名恶意的修罗场”。

这个过程在中文互联网并不罕见。
但虚拟国的坍塌，是一个特别典型的案例。
虚拟国失败了，但它的存在不是没有意义。
它代表着一群人在现实不如意的缝隙里，
仍然试图创造一个更完美世界的冲动。
只是——当这种冲动没有制度、没有边界、没有治理能力时，它很容易被戾气者带偏。

虚拟国是浪漫的，
键政是激烈的，
喷系是破坏性的。

当这三者叠在一起时，
结果从一开始就注定不是乌托邦，而是反乌托邦
